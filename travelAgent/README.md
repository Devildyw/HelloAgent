# Workflow 与 Agent区别
![img.png](img.png)

工作流：
优点：开发简单、结果确定可预测，不存在ai幻觉，性能高
缺点：每次新增规则都需要修改代码，只能处理固定的问题

Agent：
优点：灵活能够自己理解问题并处理，并不一定是预设的问题，业务适应，政策调整只需要更新知识库，基本不需要修改代码
缺点：不确定性（无法保障执行结果确定），成本高（开发成本、模型成本），延迟高
